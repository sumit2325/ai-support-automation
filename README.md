# AI-Powered Customer Support & Service Automation

> **College Project** — Frontend UI built with HTML, CSS & JavaScript

## 📋 Project Overview

An **AI-Powered Customer Support & Service Automation** system built for an international Electronics & Gadgets company. This intuitive dashboard was built to manage products, customers, orders, cases, and interactive AI chats.

> 🌟 **Live Demo:** [Click here to view the live project](https://sumit2325.github.io/ai-support-automation/)

## 🚀 Features

| Module | Description |
|--------|-------------|
| 📊 Dashboard | Real-time metrics, case overview, Agentforce activity feed |
| 👥 Customer Management | Customer records, tier management (Platinum/Gold/Silver), add new customers |
| 📦 Product Information | Electronics catalog with warranty details, ratings, stock levels |
| 🚚 Order Tracking | Real-time order status with animated progress tracker |
| 🎫 Case Management | AI auto-routing to specialized queues by category & urgency |
| 🗂️ Queue Management | Hardware, Shipping, Manager, and Agentforce AI queues with agent workloads |
| ✅ Approval Workflow | Multi-step approval process for warranties, refunds, and discounts |
| 🤖 Agentforce AI | Interactive AI chat agent that auto-handles customer queries |

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom design system with variables, Flexbox, Grid
- **JavaScript (Vanilla)** — Interactive chat, dynamic filtering
- **Font Awesome 6** — Icons
- **Google Fonts (Inter)** — Typography
